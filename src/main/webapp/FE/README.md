Frontend layer
==============

Thu muc nay chua giao dien JSP, CSS va JavaScript.

Cau truc hien tai:
- Admin/layout: header va sidebar dung chung.
- Admin/css: file giao dien.
- Admin/QuanLyHoaDon: man hinh hoa don.
- Admin/QuanLyMaGiamGia: man hinh ma giam gia.

Khi viet link trong JSP, nen tro theo context path:
${pageContext.request.contextPath}/FE/Admin/...
